Context

End User

  • Students and young professionals
  • People who want to track spending but hate filling in forms
  • Russian- and English-speaking users

Problem

Traditional expense trackers require manual data entry — picking categories, selecting dates, typing amounts. This friction causes users to abandon tracking within days.

Product Idea

Type how much you spent in a sentence — AI parses it into structured data and saves it automatically.

2 / 5

Implementation

Version 1 — Core Feature

  • FastAPI backend + PostgreSQL database
  • React frontend with Chart.js
  • AI natural language parsing (Qwen 2.5)
  • Manual expense entry form
  • Transaction history with delete
  • Spending by category chart
  • Docker Compose (4 services)

Version 2 — Templates & Polish

  • Quick-entry templates — one-click logging
  • Auto-generated default templates
  • Vendor name autocorrect
  • Groq API cloud fallback (no GPU)
  • Error handling & UI polish
